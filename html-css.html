Learn HTML & CSS - ID and Classes

First - What are HTML elements

An HTML element is part of an HTML(Hypertext Markup Language) document which tells a webpage how to structure itself.

Some elements include headers, paragraphs, text and images.
CSS (Cascading Style Sheet) is a style sheet language used to style elements in HTML documents.

CSS can be used to style (control) things like the font, colour, size of text, spacing and positioning of certains elements.

CSS selectors are used to identify certain HTML elements which you want to style.

What are ID selectors?

ID attributes allows you to select a single HTML element and modify it with styles using CSS.
IDs are unique and only targets a specific element within a page and it is identified with a hash symbol followed with the ID name.

What are class selectors?

What are the best practices associated with using classes vs. ids?

Classes should be used if there is multiple element which requires the same style to be used more than once within a page or website.

ID should be used when you only have one element on the page that is going to take on the style and can't be used more than once.

As IDs are also more specific because they only target a single element, they have a higher importance in CSS and take precedence over classes.
